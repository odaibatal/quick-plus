# مشروع كويك بلس

مشروع ويب تفاعلي عربي يتكون من خمس صفحات رئيسية تم تصميمها باستخدام Bootstrap 5 و CSS مخصص و JavaScript.

## محتوى المشروع

- `index.html` — الصفحة الرئيسية مع روابط لجميع الصفحات وخدمات تعريفية.
- `portfolio.html` — صفحة معرض الأعمال.
- `blog.html` — صفحة المدونة مع مقالات قابلة للتوسيع.
- `store.html` — صفحة المتجر مع عداد سلة تفاعلي.
- `calculator.html` — صفحة الحاسبة التفاعلية.
- `form.html` — صفحة نموذج التواصل مع تحقق من الإدخال.
- `css/style.css` — ملف CSS خارجي مخصص لتنسيق المشروع.
- `js/main.js` — وظائف تفاعلية عامة مثل تبديل الوضع الليلي وتحريك شريط التنقل.

## كيف تشغل المشروع

يمكن فتح أي صفحة HTML مباشرة في المتصفح، أو استخدام خادم محلي لتفعيل الروابط والجافاسكربت بشكل أفضل.

### تشغيل عبر Node

1. افتح مجلد المشروع في الطرفية.
2. شغل خادم محلي، على سبيل المثال:
   ```bash
   npx http-server -p 8000
   ```
3. افتح المتصفح على:
   ```
   http://127.0.0.1:8000
   ```

### تشغيل عبر Python

إذا كان Python مثبتًا:

```bash
python -m http.server 8000
```

ثم افتح الرابط:

```bash
http://127.0.0.1:8000
```
